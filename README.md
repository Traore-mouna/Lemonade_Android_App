# Lemonade_Android_App
Une application Android simple et interactive réalisée avec Jetpack Compose.
L’utilisateur doit fabriquer une limonade en suivant différentes étapes : cueillir un citron, le presser, boire la limonade puis recommencer.

# Fonctionnalités

Interface moderne avec Material 3
AppBar jaune personnalisée
Conteneur vert clair pour les images
Interaction via Modifier.clickable
Cycle complet en 4 étapes
Étape 2 configurée pour un seul tap
Utilisation de remember pour gérer les états

# L’application suit un cycle simple :

Etape 1	Cliquer sur le citron dans l’arbre
Etape 1	Presser le citron 3 Clic
Etape 3	Boire la limonade
Etape 4 Recommencer un nouveau cycle
